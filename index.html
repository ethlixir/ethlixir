Services

Trustworthiness is our founding principle

Our wizards specialize in securing smart contracts on the Ethereum blockchain and it is known to be our forte.

Coverage
Analysis

The smart contract(s) in scope are analyzed for code which is not being exercised by test cases. Missing test cases are used to improve the absolute test suite by increasing code coverage.

PENETRATION
TESTING

The smart contract(s) in scope are checked against a comprehensive database of well-known vulnerabilities. Custom-tailored attack vectors used developed to test the exploitability of the smart contract(s).

MECHANISM
DESIGN

The smart contract(s) in scope are inspected for possibilities of provoking interactive selfishness using an engineering approach. The integrity is evaluated based on behavior related to participating entities.

MANUAL
REVIEW

The smart contract(s) in scope are reviewed manually, on a source code level and line by line, to assure its quality. Knowledge transfer of host network internals, behavioral inconsistencies et al. is applied to improve code quality.

Process

Request

Send a request for an assessment which details the extensiveness of the audit. Attach the codebase subject to a security audit and associated documentation.

The codebase and the documentation will be reviewed by our team of experts who will establish a quick mental model of the complexity of the request.

Quote

Our team will send you an estimated quote based on the audit extensiveness, code complexity and project variables related to the request.

Auditing

Our hackers are put to work. Multiple hackers are used for each assessment to decrease the time of delivery.

The team will audit the codebase on an individual basis and then come together in a collaborative effort to share their security issues and form a thorought audit.

Report

Our team will deliver an extensive deliverable with all security issues and recommendations on how to fix the observations.
